# ClawPrompt Chrome Web Store Submission

## Extension Overview

**Name:** ClawPrompt - AI Prompt Templates
**Version:** 1.0.0
**Category:** Developer Tools
**Description:** Manage and insert AI prompt templates into any text field. Perfect for Claude, ChatGPT, and other AI assistants.

## Features

- Template management with categories, tags, and favorites
- Quick search and filtering
- Context menu integration
- Keyboard shortcuts ( Ctrl+Shift+P to open, Ctrl+Shift+I for quick insert)
- Import/Export templates
- Template variables ( {{cursor}}, {{selected}}, {{date}}, etc.)
- Dark theme with modern UI
- Offline storage ( all data stays local)

## Installation

### From Source
1. Clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the `extensions/clawprompt` folder

## Usage

1. **Open Popup**: Click the ClawPrompt icon in Chrome toolbar
2. **Search Templates**: Use the search bar to find templates
3. **Insert Template**: Click on a template to insert it into the active text field
4. **Quick Insert**: Press `Ctrl+Shift+P` for quick template picker
5. **Right-Click**: Use context menu on any text field
6. **Options Page**: Right-click extension icon → Options for full management

## Development

```bash
# Build extension
cd extensions/clawprompt
chmod +x build.sh

# This creates:

```

## Chrome Web Store Submission

1. Create a developer account at [Chrome Web Store Developer Dashboard](https://chrome.google.com/webstore/devconsole)
2. Click "Add new item"
3. Upload the `clawprompt-extension.zip`
4. Fill in the store listing details:
5. Upload screenshots (1280x800 or 640x400)
7. Submit for review

## Screenshots Required
- 1280x800: Main popup interface
- 1280x800: Options page
- 640x400: Small promo tile
