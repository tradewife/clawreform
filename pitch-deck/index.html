Key Innovation #1

The Organ System

Agent context as discrete, human-editable, git-trackable files — not a prompt blob. Injected into the runtime in deterministic order.

IDENTITY.mdMission & values
SOUL.mdTone & character
HANDS.mdAction doctrine
MEMORY.mdMemory law
HEARTBEAT.mdAutonomy cadence
TOOLS.mdTool constraints
SKILLS.mdLearned methods
USER.mdHuman model

Edit SOUL.md in any text editor → agent personality changes instantly. Version it in git → full audit trail.

Key Innovation #2

The Memory Ladder

Four-tier memory with biological decay. Important things persist — the rest fades naturally.

Core

Durable truths. High edit friction. Survive across projects and time.

Overview

Strategic map of current priorities. Updated when direction changes.

Projects

Auto-curated ledgers per initiative. Promoted from validated detail.

Working Detail

Hot context. Aggressively compacted. LLM-summarized, not truncated.

+ Cross-channel canonical sessions: tell it on Telegram, it knows on Discord.

06
Key Innovation #3

Self-Modification Engine

ClawReform can rewrite its own Rust source code — with a full safety pipeline.

1. Analyze

Parse request, identify affected files, generate a plan.

2. Backup

Snapshot every affected file before touching anything.

3. Modify

Apply changes under max_files_per_mod safety limits.

4. Validate

Run cargo build, cargo test, clippy on modified source.

5. Rollback

Auto-restore from backup if validation fails. Zero data loss.

6. Approve

Integration with approval workflows — human-in-the-loop optional.

07
Production Engineering

Built for reliability,
not demos

Circuit Breaker

Per-provider cooldown prevents request storms. Exponential backoff with intelligent error classification.

Context Budgeting

Dynamic overflow recovery pipeline. Tool results truncated proportionally. LLM-based session compaction.

Loop Guard

Detects repetitive tool calls. Configurable circuit breaker. Prevents infinite loops and runaway costs.

Session Repair

Auto-repairs orphaned tool results. Merges consecutive messages. Recovers text-based tool calls from non-standard models.
